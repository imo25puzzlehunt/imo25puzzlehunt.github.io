Solution: Tri-Alphametic
Answer: GOLD MEDAL

Written by Ross Atkins

An alphametic or cryptarithm puzzle is a puzzle where you are given a sum whose digits have been replaced with different letters, such as SEND+SOME=MONEY, and the goal is to recover the original equation.

In this puzzle, we are given three sums which share the same set of letters; by using logical deduction and all three sums we can deduce what each letter is: A=0, E=1, D=2, O=3, T=4, M=5, H=6, L=7, G=8 and I=9.

By itself this doesn’t spell out anything, but we are also given some digits at the bottom. Replacing them with the correct letters, we get the answer, GOLD MEDAL.
